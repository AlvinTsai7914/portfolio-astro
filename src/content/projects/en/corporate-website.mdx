## Project Overview

A corporate website built for a tech startup, featuring a clean design and smooth browsing experience. Supports bilingual Chinese/English switching with comprehensive SEO optimization.

## My Role

Independently handled the complete process from design to development:

- Built a static website using the Astro framework
- Implemented route-based i18n bilingual system
- Created modular SCSS styling architecture
- SEO meta tags, Open Graph, and structured data

## Technical Highlights

### Astro Islands Architecture

JavaScript is only loaded for interactive components; everything else is static HTML. The homepage JavaScript bundle is only 12KB (gzipped), achieving exceptional loading speed.

### i18n Routing Design

Path-based bilingual routing (`/zh/`, `/en/`), with each page having its own URL for better SEO indexing. Language switching preserves the current page position for a seamless user experience.

## Lessons Learned

My first complete project using the Astro framework, where I experienced the tremendous advantages of Islands Architecture for content-focused websites.
