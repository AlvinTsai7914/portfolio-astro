## Project Overview

A data dashboard system developed for an internal operations team, transforming complex business data into intuitive charts and reports. Supports real-time data updates, custom date range filtering, and PDF/CSV report export.

## My Role

Frontend development, responsible for:

- Implementing multiple chart types with Chart.js (line, bar, pie)
- Building a reusable Dashboard Widget component system
- Implementing real-time data update mechanism (WebSocket + polling fallback)
- Report export functionality (frontend PDF generation)

## Technical Highlights

### Chart Component System

Encapsulated Chart.js as React components supporting theme switching, responsive adjustments, and animations. Different chart types can be quickly created through props, significantly reducing development costs.

### Data Caching Strategy

Used React Query for server state management, implementing smart caching and background revalidation to reduce API requests while ensuring data freshness.

## Lessons Learned

Understood the design principles of data visualization — it's not just about "drawing data as charts," but choosing the right chart types to convey information and handling performance issues with large datasets.
