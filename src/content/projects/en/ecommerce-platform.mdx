## Project Overview

A full-featured e-commerce platform providing a complete online shopping experience. From product browsing and cart management to secure checkout, every step is carefully designed to ensure a smooth user experience.

## My Role

As the lead frontend developer, I was responsible for the entire frontend architecture and implementation, including:

- Product listing and filtering system
- Cart state management (React Context + useReducer)
- Responsive layout design (mobile-first)
- Performance optimization (lazy loading, code splitting)

## Technical Highlights

### Component Architecture

Adopted the Atomic Design methodology, breaking down the UI into reusable atomic components to build a consistent design system. All components are written in TypeScript for type safety.

### Performance Optimization

- Images use WebP format + lazy loading, improving homepage load speed by 40%
- Used React.memo and useMemo to reduce unnecessary re-renders
- Lighthouse performance score reached 95+

### Accessibility

- All interactive elements support keyboard navigation
- Semantic HTML tags throughout
- Color contrast meets WCAG 2.1 AA standards

## Lessons Learned

This project deepened my understanding of complex e-commerce business logic, especially in handling inventory synchronization, pricing calculations, and order state management.
