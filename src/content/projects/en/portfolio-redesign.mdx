## Project Overview

A complete redesign of my personal portfolio website, with the core goal of showcasing frontend development capabilities. Combines minimalist design with refined dynamic effects, balancing visual expression with technical depth.

## My Role

Independently completed the entire process from design concept to deployment:

- Design research referencing outstanding portfolio websites
- Built with Astro framework using Islands Architecture
- GSAP + ScrollTrigger for scroll animations
- MDX content management (project Case Studies)

## Technical Highlights

### Design System

Built a complete Design Token system (colors, typography, spacing, border-radius), implementing a visual rhythm of alternating light/dark sections through CSS Custom Properties.

### Scroll Animations

Used GSAP ScrollTrigger for element entrance animations. Every animation is intentional — conscious motion, not animation for animation's sake.

### Performance

- Static generation, zero JavaScript overhead (except for interactive components)
- Fonts use `font-display: swap` to avoid FOIT
- Image optimization + lazy loading

## Lessons Learned

This project taught me that "design" and "development" are not two separate phases, but a continuous iterative process. A good portfolio doesn't just showcase technical skills — it demonstrates problem-solving mindset.
