## Overview

A personal practice project — cloning the Twitch responsive homepage with React and the Twitch API, used as a vehicle to study large-site structure and component design.

## My Role

Solo developer, responsible for the full frontend implementation.

## Technical Highlights

- **Component thinking**: Decomposed the homepage into reusable components (GameCard, StreamerCard, Sidebar, etc.) controlled via props
- **State management**: React state for API data, loading states, and interaction states
- **API integration**: Integrated two Twitch APIs (Top Games + Streams by Game ID) to surface "top games by viewership" and "top streamers per game"
- **styled-components**: CSS-in-JS to reproduce Twitch's visual and interaction details
- **Responsive**: Desktop and mobile support
