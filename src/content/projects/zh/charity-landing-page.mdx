## 專案概述

公益平台一頁式形象網站，重點呈現組織理念、服務項目與聯繫方式。

## 我的角色

前端工程師，負責：

- 全站 HTML / SCSS / Bootstrap 切版
- Slick.js 圖片輪播實作
- CSS Animation 背景動畫
- Git 版本控管

## 技術重點

- **純 CSS 動畫**：背景互動效果以 CSS Animation 實作，避免 JS 開銷
- **Slick.js slider**：響應式輪播、自動播放、觸控支援
- **Bootstrap RWD**：手機優先佈局，跨裝置相容
