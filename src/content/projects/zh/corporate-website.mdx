## 專案概述

為一間科技新創公司打造的企業形象網站，主打簡潔的設計風格與流暢的瀏覽體驗。支援中英雙語切換，並針對搜尋引擎進行全面最佳化。

## 我的角色

獨立負責從設計到開發的完整流程：

- 使用 Astro 框架建構靜態網站
- 實作路由式 i18n 雙語系統
- 建立 SCSS 模組化樣式架構
- SEO meta tags、Open Graph、結構化資料

## 技術亮點

### Astro Islands Architecture

只在需要互動的元件載入 JavaScript，其餘全部為靜態 HTML。首頁 JavaScript bundle 僅 12KB（gzipped），達到極致的載入速度。

### i18n 路由設計

採用路徑式雙語（`/zh/`、`/en/`），每個頁面都有獨立的 URL，有利於 SEO 索引。語言切換時保持當前頁面位置，使用者體驗無縫銜接。

## 學到的事

第一次使用 Astro 框架開發完整專案，體會到 Islands Architecture 在內容型網站的巨大優勢。也更深入理解了 SEO 的技術面向，包括 Core Web Vitals 的最佳化策略。
