## 專案概述

為內部營運團隊開發的數據儀表板系統，將複雜的業務數據轉化為直觀的圖表和報表。支援即時資料更新、自訂日期範圍篩選，以及 PDF/CSV 報表匯出。

## 我的角色

前端開發，負責：

- 使用 Chart.js 實作多種圖表（折線圖、長條圖、圓餅圖）
- 建立可複用的 Dashboard Widget 元件系統
- 實作即時資料更新機制（WebSocket + polling fallback）
- 報表匯出功能（前端生成 PDF）

## 技術亮點

### 圖表元件系統

封裝 Chart.js 為 React 元件，支援主題切換、響應式調整和動畫效果。透過 props 即可快速建立不同類型的圖表，大幅降低開發成本。

### 資料快取策略

使用 React Query 管理伺服器狀態，實作智慧快取和背景重新驗證，減少 API 請求次數同時確保資料即時性。

## 學到的事

理解了資料視覺化的設計原則——不只是「把資料畫成圖表」，更重要的是選擇正確的圖表類型來傳達資訊，以及如何處理大量資料的效能問題。
