## 專案概述

這是一個全功能的電商平台，為客戶提供完整的線上購物體驗。從商品瀏覽、購物車管理到安全結帳，每個環節都經過精心設計，確保流暢的使用者體驗。

## 我的角色

作為前端開發主力，負責整個前端架構設計與實作，包括：

- 商品列表與篩選系統
- 購物車狀態管理（React Context + useReducer）
- 響應式佈局設計（mobile-first）
- 效能最佳化（lazy loading、code splitting）

## 技術亮點

### 元件架構

採用 Atomic Design 方法論，將 UI 拆解為可複用的原子元件，建立一致的設計系統。所有元件都使用 TypeScript 撰寫，確保型別安全。

### 效能最佳化

- 圖片使用 WebP 格式 + lazy loading，首頁載入速度提升 40%
- 使用 React.memo 和 useMemo 減少不必要的重渲染
- Lighthouse 效能分數達到 95+

### 無障礙設計

- 所有互動元素皆支援鍵盤操作
- 使用語意化 HTML 標籤
- 色彩對比度符合 WCAG 2.1 AA 標準

## 學到的事

這個專案讓我深入理解了電商領域的複雜業務邏輯，特別是在處理庫存同步、價格計算和訂單狀態管理方面。也讓我更熟練地運用 TypeScript 的進階型別來建模複雜的資料結構。
