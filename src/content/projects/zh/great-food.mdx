## 專案概述

與朋友合作的餐飲品牌網站。原始版本是全圖片式網頁（不利搜尋與閱讀），重新以 HTML + CSS + Bootstrap 切版，提升使用者瀏覽體驗與搜尋引擎索引能力。

## 我的角色

前端工程師，負責切版與 RWD 實作。

## 技術重點

- **語意化轉換**：將全圖片式內容轉為文字 + 圖片混合，方便搜尋引擎索引
- **Bootstrap 排版**：使用 Bootstrap Grid 系統處理多斷點佈局
- **RWD 切版**：手機、平板、桌機三斷點全支援
