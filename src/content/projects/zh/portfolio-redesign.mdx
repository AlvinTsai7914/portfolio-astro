## 專案概述

個人作品集網站的全面重新設計，以展現前端開發能力為核心目標。採用極簡設計風格搭配精緻的動態效果，在視覺表現與技術深度之間取得平衡。

## 我的角色

從設計概念到開發部署，全程獨立完成：

- 參考優秀作品集網站進行設計研究
- 使用 Astro 框架搭配 Islands Architecture
- GSAP + ScrollTrigger 實作滾動動畫
- MDX 內容管理（專案 Case Study）

## 技術亮點

### 設計系統

建立完整的 Design Token 系統（色彩、字體、間距、圓角），透過 CSS Custom Properties 實現深淺區塊交替的視覺節奏。

### 滾動動畫

使用 GSAP ScrollTrigger 實作元素進場動畫，每個動畫都是刻意設計的——有意識的動態，而非為動畫而動畫。

### 效能

- 靜態生成，零 JavaScript 開銷（除互動元件外）
- 字體使用 `font-display: swap` 避免 FOIT
- 圖片最佳化 + lazy loading

## 學到的事

這個專案讓我體認到「設計」和「開發」不是兩個獨立的階段，而是不斷迭代的過程。一個好的作品集不只是展示技術，更是展示解決問題的思維方式。
