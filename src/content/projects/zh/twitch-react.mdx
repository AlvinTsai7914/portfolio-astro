## 專案概述

個人練習專案，使用 React 與 Twitch API 複刻 Twitch RWD 首頁，藉此學習大型網站的結構與元件設計邏輯。

## 我的角色

獨立開發，負責全部前端實作。

## 技術重點

- **元件化思維**：將首頁拆解為可複用的元件（GameCard、StreamerCard、Sidebar 等），用 props 控制變化
- **狀態管理**：以 React state 管理 API 資料、loading 狀態與互動狀態
- **API 串接**：串接兩支 Twitch API（Top Games + Streams by Game ID），組合資料呈現「最高觀眾遊戲」與「各遊戲最高實況主」
- **styled-components**：CSS-in-JS 重現 Twitch 的視覺與互動細節
- **RWD 切版**：支援桌機與行動裝置
